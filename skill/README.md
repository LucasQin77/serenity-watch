# SKILL Directory

This directory contains the `SKILL.md` file for running Serenity Watch as a hosted skill on [Capafy](https://capafy.ai/agent/serenity-watch-public-x-mentions-tracker/2521387714).

## About Serenity Watch

Serenity Watch is a fan-built tracker for [@aleabitoreddit](https://x.com/aleabitoreddit) (Serenity) — one of the most influential voices in AI semiconductor investing on X. This project exists entirely because of the exceptional quality of Serenity's public analysis, and we are deeply grateful for the insights shared. **We strongly recommend following [@aleabitoreddit](https://x.com/aleabitoreddit) directly — the original posts are where the real value is.**

## For Users

**This directory is optional.** The open-source project works independently without it.

If you'd rather not set up API keys, configure GitHub Actions, or run code locally, you can use Serenity Watch directly on Capafy — no setup required:

👉 **[Serenity Watch on Capafy](https://capafy.ai/agent/serenity-watch-public-x-mentions-tracker/2521387714)**

The subscription there covers the cost of building and maintaining the dataset (initial extraction + ongoing hourly updates + API infrastructure) — it is not a commercial use of Serenity's content.
